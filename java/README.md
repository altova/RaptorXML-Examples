# RaptorXML-Examples in C# #

Examples for using the Java API of [RaptorXML+XBRL Server](http://www.altova.com/raptorxml.html).

#### Xbrl_LoadAndValidate  
These example console applications show how to load an XBRL instance, check for validation errors and perform one small additional feature for demonstration purposes.

##### Xbrl_CalculationLinkbaseTraversal
* traverse the calculation linkbase arcs

##### Xbrl_DimensionalRelationshipSetTraversal
* traverse the dimensional relationship set

##### Xbrl_FindDuplicateContexts
* search for duplicate contexts

##### Xbrl_FindDuplicateUnits
* search for duplicate units

##### Xbrl_FindFactsByName
* search for specific facts (by concept name)

##### Xbrl_FindFactsWithSameAspects
* search for specific facts (by concept name) that share the same aspects

##### Xbrl_InstanceStatistics
* access the data model of the XBRL instance and the supporting DTS

##### Xbrl_PresentationLinkbaseTraversal
* traverse the presentation linkbase arcs

##### Xslt_Test
* Introductory example for XSLT engine usage via the Java API

##### XPath_GettingStarted
* Introductory example for XPath engine usage via the Java API

##### XPathInitialContextXmlDocument
* Introductory example for XPath engine usage via the Java API

##### XPath_ExternalFunctions
* Example for user-written functions called from XPath expressions

##### XQuery_ExternalFunctions
* Example for user-written functions called from XQuery expressions
